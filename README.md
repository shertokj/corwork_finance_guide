# Claude Cowork — Finance Prospecting Guide

A practitioner's field guide for the **Corporate Cash Investment SVP** on using Claude Cowork, agent skills, scheduled routines, and live artifacts to run a modern, AI-assisted prospecting practice.

Built on Claude Cowork GA, the February 2026 plugin expansion, and the May 2026 financial services agent launch.

## What's Inside

The guide covers six core areas and three detailed appendices:

### Main Guide
| Section | Description |
|---|---|
| The Lay of the Land | What Cowork is, how its five layers work together |
| Setup | Plans, plugins, connectors, Claude in Chrome, project context |
| Prospecting Workflows | Four on-demand workflows with copy-paste prompts |
| Scheduled Tasks | Recurring task architecture and scheduling logistics |
| Boundaries | What Claude won't do, LinkedIn rules, compliance notes |
| Starting Stack | Recommended tools and cost breakdown |

### Appendix A — Agent Skills
Seven skills with exact creation prompts, covering:
- Treasury Contact Classifier (Tier 1/2/3 framework)
- Pre-Call Research Protocol (six-section structured brief)
- Outreach Voice and Compliance Guard (tone rules + auto-flagging)
- Counterparty Credit Monitor (materiality classification)
- Conference Intelligence Extractor (speaker directory extraction)
- Market Rate and Instrument Digest (standardized rate data format)
- Relationship Status Tracker (pipeline taxonomy + Gmail scanning)

### Appendix B — Routines Library
Fourteen named routines covering the full operating cycle:
- Daily: Morning Market Open
- Weekly: Coverage List Refresh, New Hire Radar, Regulatory Watch
- Biweekly: Pipeline Follow-Up Audit
- Monthly: New Coverage Expansion, Counterparty Credit Sweep, Outreach Performance Review
- Quarterly: Relationship Review, Dormant Re-Engagement Campaign
- On-demand: Conference Season Scan, Earnings Season Cash Watch, Fed Decision Briefing
- Annual: Coverage Audit

### Appendix C — Live Artifacts
Eight persistent, data-connected artifacts that form the operational infrastructure:
- Master Coverage Sheet (central contact database)
- Outreach Queue (draft staging with compliance flags)
- Call Brief Library (Drive folder structure)
- Active Counterparty Watchlist (credit monitoring scope)
- Credit Monitor Log (continuously appended event log)
- Market Rate Tracker (rolling rate data with charts)
- New Hire Radar Log (hire detection with outreach timing)
- Relationship Review Report (quarterly/annual template)
